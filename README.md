# AlgoritimoGenetico em Java Puro

Este projeto é uma implementação de um algoritmo genético simples, desenvolvida para uso em ambiente acadêmico.  
O objetivo é mostrar, de forma prática e clara, como funciona um algoritmo genético com as etapas principais (população, seleção, cruzamento, mutação, avaliação).

---

## Tecnologias Utilizadas  
- Java (sem frameworks externos)  
- Ambiente local (“computador da faculdade”, recursos limitados)  
- Lógica de algoritmo genético implementada do zero  

---

## Objetivo do Projeto  
- Demonstrar a estrutura básica de um algoritmo genético  
- Implementar população inicial, função de aptidão, operadores de seleção, cruzamento e mutação  
- Verificar evolução da solução ao longo de gerações  
- Usar ambiente simples e código direto, sem dependências complexas  

---

## Como Executar  
1. Abra o projeto no seu ambiente Java (IDE ou linha de comando).  
2. Compile os arquivos `.java`.  
3. Execute a classe principal (ex: `Main.java`).  
4. Observe os resultados no console: aptidão ao longo das gerações, melhor indivíduo, etc.

---

## Estrutura do Código  
- Classe principal que inicializa população e gerações  
- Classe “Indivíduo” ou “Cromossomo” que representa soluções  
- Métodos para: avaliação (função de aptidão), seleção, cruzamento (crossover), mutação  
- Loop de gerações que atualiza população até critério de parada  

---

## Observações  
- Projeto voltado para fins acadêmicos e didáticos  
- O ambiente usado era de recursos modestos, então o código foi escrito para ser simples, direto e leve  
- Pode ser estendido com diferentes funções de aptidão, representação de cromossomos, problemas mais complexos
